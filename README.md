 Projeto: Venda de móveis
 Alunos: Eduardo Bergamo, Lívia Reinheimer, Kauã Lima Santos
 Objetivo: Tem como função automatizar o cálculo de orçamento para clientes de uma loja de móveis, facilitando a venda.
 Apresenta móveis disponíveis com preço fixo através de um menu numerado, focado em agilizar o atendimento e padronizar os desconots.
